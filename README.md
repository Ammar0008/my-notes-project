My Notes Project

A full-stack note-taking app built with Java (Spring Boot) backend and a frontend interface. Developed for learning Java, Maven, and Jenkins pipelines.

Features

CRUD operations for notes via REST API

Backend: Java 21, Maven

Frontend: HTML, JS, CSS

Tested with Jenkins CI/CD pipeline

Run

Backend:

mvn clean install
mvn exec:java


Runs at: http://localhost:8081/notes

Frontend:
Open frontend files in a browser or IDE.
